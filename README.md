# Kaelora Backend

Backend API for the Kaelora E-commerce Platform.

## 🚀 Features

* User Authentication & Authorization
* Admin Dashboard API
* Product Management
* Category Management
* Cart System
* Order Management
* Guest Order Tracking
* Payment Integration
* Image Upload with Cloudinary
* JWT Authentication
* MongoDB Database

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* Multer
* Cloudinary

---



## 📸 Image Upload

This project uses Cloudinary for image storage.

---

## 🔐 Authentication

JWT based authentication system is implemented.

Protected routes require Bearer Token.

Example:

```bash
Authorization: Bearer your_token
```

---

## 🚀 Deployment

Backend deployed on Render.
